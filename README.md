<p align="center">

 ```python -m venv [your_environment_name]```
 
 **windows**
 
 ```
 .\myenv\bin\activate [your_environment_name]

 ```
 
 **linux**
 
 ```
 source myenv/bin/activate
 pip install -r requirements.txt
 ```
